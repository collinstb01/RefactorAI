# RepoAudit - AI GitHub Analysis Dashboard

RepoAudit is a modern developer SaaS dashboard that analyzes GitHub repositories using AI to provide insights about architecture, technical debt, and code patterns.

## Features

- **Landing Page**: High-conversion hero section and feature highlights.
- **Dashboard**: Overview of connected repositories with tech debt scoring.
- **Repo Analysis**: Deep dive into code complexity, architecture, and suggested improvements.
- **Settings**: Manage AI models and GitHub connections.

## Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Charts**: Recharts
- **Animations**: Framer Motion

## Getting Started

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
